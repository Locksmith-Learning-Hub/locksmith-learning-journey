# 🛠️ Module 6: Tools Familiarisation

## 📌 Overview
This module introduces the essential tools used in locksmith work. Understanding the purpose, handling, and limitations of each tool is important for safe, efficient, and professional performance in real-world situations.

---

## 🔑 Lock Pick Sets (Overview)

Lock pick sets are used for non-destructive entry techniques.

### 🔍 Common Components:
- Hook picks (for precise pin control)  
- Rake picks (for faster entry on simple locks)  
- Tension wrench (applies controlled turning pressure)  

### 🧠 Key Understanding:
- Picks are used to manipulate internal pins  
- The tension wrench creates a binding effect  
- Both tools must be used together with control  

> ⚠️ Skill and patience are required — improper use can damage the lock.

---

## 🔧 Air Wedge and Entry Tools

Air wedges are commonly used for safe entry, especially in vehicle lockouts.

### 🛠️ Air Wedge:
- Inserted between door and frame  
- Inflated to create a small gap  
- Allows access for other entry tools  

### 🔍 Long Reach Tools:
- Used to unlock doors from inside  
- Common in automotive lockouts  

> 💡 These tools allow entry without damaging the door or lock.

---

## 🔩 Basic Hand Tools

Every locksmith relies on a set of basic hand tools for installation and repair work.

### 🔧 Screwdrivers
- Used for removing and fixing screws  
- Essential for lock replacement and fitting  

### ⚙️ Drills
- Used when destructive entry is required  
- Helps remove damaged or high-security locks  

> ⚠️ Drilling should be used only when non-destructive methods are not possible.

---

## 🧰 Tool Usage Principles

- Use the right tool for the right task  
- Avoid excessive force  
- Maintain tools in good condition  
- Always test results after using tools  

---

## ⚠️ Safe Handling of Tools

Safety is critical in locksmith work.

### 🛡️ Best Practices:
- Keep tools organized and accessible  
- Use protective equipment when required  
- Handle sharp tools carefully  
- Avoid rushing during operations  

> 💡 Safe handling reduces risk and improves efficiency.
